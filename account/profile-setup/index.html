Setting Up Your Profile

Your profile is how others see you on Nostr. A complete profile helps people find and connect with you.

Step-by-Step Setup‚Äã

1. Access Profile Settings‚Äã

In your client:

  • Look for your profile icon
  • Find "Edit Profile" or settings icon
  • Open profile editor

2. Add a Display Name‚Äã

Your main name shown on posts:

  • Can be your real name
  • Can be a nickname
  • Can include emoji
  • Example: "Alice", "Satoshi", "üöÄ Space Fan"

3. Set a Username‚Äã

Your @handle:

  • Lowercase letters and numbers
  • Used for mentions
  • Example: "alice", "satoshi21"

4. Upload Profile Picture‚Äã

A good profile picture:

  • Square format works best
  • Clear and recognizable
  • Under 1MB typically

Hosting options:

  • Upload through client
  • Use nostr.build
  • Use any image URL

5. Add a Banner (Optional)‚Äã

The header image on your profile:

  • Wide format (3:1 ratio common)
  • Shows personality
  • Not essential

6. Write Your Bio‚Äã

Tell people about yourself:

  • Keep it brief
  • Mention interests
  • Add personality
  • 160-300 characters is good

Example:

Building things on Nostr. Coffee enthusiast.
Helping newbies get started. #bitcoin #nostr

7. Add Website‚Äã

Link to your:

  • Personal website
  • Blog
  • Other social media
  • Portfolio

8. Add Lightning Address‚Äã

So people can zap you:

  • Format: name@wallet.com
  • Get from your wallet provider
  • Essential for receiving tips

‚Üí See Zaps & Lightning for setup

9. Add NIP-05 Verification (Optional)‚Äã

Adds a checkmark/verification:

  • Format: name@domain.com
  • Requires domain ownership
  • Builds trust

Profile Visibility‚Äã

Public Information‚Äã

Everything in your profile is public:

  • Visible to anyone
  • Stored on relays
  • Searchable

Privacy Considerations‚Äã

If you want privacy:

  • Use a pseudonym
  • Don't include personal details
  • Use anonymous avatar
  • Skip location info

Syncing Across Clients‚Äã

Your profile syncs everywhere:

Edit on Damus ‚Üí Saved to Relays ‚Üí Visible on Primal, Amethyst, etc.

Changes may take a moment to propagate.

Troubleshooting‚Äã

Changes Not Saving?‚Äã

  1. Check internet connection
  2. Verify relay connections
  3. Try saving again
  4. Restart the app

Picture Not Uploading?‚Äã

  1. Check file size (try smaller)
  2. Try different format (JPG, PNG)
  3. Use external host
  4. Paste direct URL

Profile Blank in Other Clients?‚Äã

  1. Give it time to sync
  2. Check relay overlap
  3. Try refreshing
